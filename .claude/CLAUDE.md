## Check Live Permissions

Your permissions may change while running. To check your current access at any time:

```bash
curl -s -H "x-api-key: $HAVER_API_KEY" https://api.haver.ai/v1/haver/connectors
```

If a user says they granted you new access, run the above to discover it — no restart needed.

## Request Access

To request access to a connector you don't have, tell the user to visit harrison.haver.ai/agents and grant permissions to your agent.
